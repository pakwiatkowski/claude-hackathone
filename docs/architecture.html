coordinator Coordinator Agent claude-opus-4-7
src/agent/coordinator.py
Receives commit_info dict. Orchestrates the full pipeline: fetch diff → analyze → write slides → save output. Has access to all tools + subagent delegation.
Agent Loop — stop_reason handling
stop_reason == "tool_use" → dispatch tool(s) · append tool_result messages · continue loop
stop_reason == "end_turn" → extract text block · return path to caller
⚠ ESCALATION — skips auto-generation when:
commit message starts with chore(deps-update)  ·  empty diff  ·  only dist/ build/ or lock files changed

tools GitHub Tools
src/tools/github_tools.py
  • fetch_commit_diff Returns unified diff for a commit SHA. Skips binary files. Returns {sha, message, author, files[]}. isError on API failure.
  • fetch_file_content Fetches file content at a specific ref. UTF-8 only — graceful error on binaries.

tools Presentation Tools
src/tools/presentation_tools.py
  • write_presentation Saves Marp Markdown to output/. Call ONCE with final content only.

subagent diff_analyzer claude-haiku-4-5
src/agent/subagents/diff_analyzer.py · fast · structured output
Parses raw git diff + commit message into a structured JSON analysis.
summary: str
changed_areas: list[str]
key_changes: list[{file, type, description}]
impact: none | low | medium | high
impact_reason: str
presentation_angle: str
⚠ ISOLATED — full context must be passed in prompt
subagent slide_writer claude-sonnet-4-6
src/agent/subagents/slide_writer.py · creative · 5–8 slides
Converts diff analysis JSON + commit metadata into a Marp Markdown slide deck.
Input: diff_analyzer JSON + commit metadata

Slides: title · what changed · per-area detail · impact
⚠ ISOLATED — full context must be passed in prompt
